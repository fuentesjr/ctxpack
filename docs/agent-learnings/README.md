# Agent learnings

Durable learning notes written by the `extract-approach` skill
(`.agents/skills/extract-approach/SKILL.md`) after every non-trivial solved
problem. One note per file: `YYYY-MM-DD-short-title.md`, with the eight
sections the skill defines (problem, context, failed approaches, key insight,
final approach, verification, reusable rule, when to apply again).

Search here before re-debugging anything that smells familiar.
